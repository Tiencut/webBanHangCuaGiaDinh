Hướng dẫn kiểm tra Navigation

1. Kiểm tra Dropdown Menu

Hover vào menu "Hàng hóa" ở trên để thấy dropdown menu với các tùy chọn:

  • Danh sách hàng hóa
  • Thiết lập giá

2. Kiểm tra Link Trực tiếp

Nút "Trực tiếp → Hàng hóa" màu xanh sẽ dẫn trực tiếp đến trang Products

3. Khắc phục sự cố

Nếu dropdown không hoạt động:

  • Kiểm tra CSS hover effects
  • Đảm bảo không có JavaScript conflict
  • Sử dụng link trực tiếp thay thế
