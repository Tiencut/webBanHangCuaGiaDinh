  3. 3. Click lại thấy ▼ (Z-A)
  4. 4. Thử với cột "Giá bán"

🔍 Test Search

  1. 1. Nhập "lọc" vào search box
  2. 2. Thấy kết quả lọc real-time

  2. 2. Thấy chỉ hiện sản phẩm động cơ
